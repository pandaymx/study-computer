# CPU

## 简介

### 什么是 CPU

CPU 的全程是 Central Processing Unit，中文称为中央处理器。它是计算机的核心部件之一，负责执行计算机程序中的指令，并进行算术和逻辑运算。

### CPU 的作用

CPU 的主要作用是处理数据和指令。它从内存中读取指令，解码并执行这些指令，然后将结果存储回内存或输出到其他设备。CPU 的性能直接影响计算机的运行速度和效率。

### CPU 的重要性

CPU 是计算机系统的核心组件之一，几乎所有的计算任务都需要通过 CPU 来完成。它的性能和架构直接决定了计算机的处理能力和响应速度。

## CPU 性能指标

### 架构

CPU 的内部设计蓝图，每一代都有可能带来巨大的性能提示，因此对 CPU 的挑选来说，新一代的 CPU 可能有更大的优势。

### 主频（时钟频率）

主频是指 CPU 每秒钟可以执行的指令数，通常以 GHz（千兆赫兹）为单位。主频越高，CPU 的处理速度通常也越快。

但是，主频并不是衡量 CPU 性能的唯一指标。其他因素如核心数、缓存大小和指令集架构等也会影响 CPU 的实际性能。

:::danger[请不要认为主频越强越好]

主频越强虽然越厉害，但受限于物理因素，主频的提升会带来更高的功耗和热量，因此现代 CPU 在设计时会综合考虑多种因素来平衡性能和功耗。

:::

### 核心数

核心数是指 CPU 中独立的处理单元数量。多核心 CPU 可以同时处理多个任务，提高计算机的多任务处理能力。

核心数越多，CPU 处理多任务的性能就越强，一般而言，核心数是衡量 CPU 性能的重要指标之一。

:::tip[大小核技术]

Intel 引入性能核和效率核的大小核技术，AMD 也在其最新的 CPU 中采用了类似的设计。对于游戏来说，性能核负责处理高负载任务，而效率核则处理低负载任务，从而提高整体性能和能效。

但由于优化不当、适配不佳等原因，大小核技术可能会导致实际体验效果不如预期，尤其是在游戏中可能会出现性能瓶颈。

更多详情请参见文章：[CPU 大小核技术探讨](/blog/2025/08/02/cpu-big-little)

:::

### 超线程

超线程技术允许每个物理核心模拟出两个逻辑核心，从而提高 CPU 的并行处理能力。通过超线程，CPU 可以在同一时间处理更多的线程，提高多任务处理效率。

但是超线程技术可能会导致游戏性能下降，因为部分游戏对多线程支持不佳，可能会出现性能瓶颈。

最新的 intel CPU 开始逐步取消超线程技术，转而采用最新的大小核技术。

### 缓存

缓存是 CPU 内部的一种高速存储器，用于临时存储频繁使用的数据和指令。CPU 缓存分为 L1、L2 和 L3 三级，L1 缓存速度最快但容量最小，L3 缓存速度较慢但容量较大。缓存的大小和速度对 CPU 性能有重要影响。

缓存是越大越好，但一般考虑到成本和功耗，CPU 的缓存大小不会太大。

:::tip[X3D 技术]

X3D 技术是 AMD 在其 Ryzen 系列 CPU 中引入的一种新型缓存架构。通过将 L3 缓存分为多个小块并采用 3D 堆叠技术，X3D 技术可以显著提高缓存的访问速度和带宽，从而提升 CPU 的整体性能。

:::

### TDP（热设计功耗）

TDP 是指 CPU 在正常工作状态下的最大热输出功率，通常以瓦特（W）为单位。TDP 反映了 CPU 的功耗和散热需求，影响系统的散热设计和能效。

### 指令集架构（ISA）

指令集架构是 CPU 能够理解和执行的指令集合。不同的 CPU 可能支持不同的指令集架构，如 x86、ARM 等。指令集架构决定了 CPU 的功能和性能特性。

## CPU 命名规则

### AMD 

**主要特点**

多核性能优越，无论生产力应用还是游戏性能都表现出色。AMD 的 CPU 通常具有更高的性价比，尤其是在多线程任务中表现更佳。

服务周期长，未来升级空间大，AMD 的 CPU 通常支持更长的服务周期，用户可以在较长时间内享受性能提升和新技术的支持，主板支持多代 CPU。

**命名规则**

以 AMD Ryzen 7 7800X3D 为例：
- Ryzen：品牌名称。
- 7：产品定位，数字越大性能越强（Ryzen 3 → 5 → 7 → 9）。
- 7 (第一位数字)：代数，代表第7代锐龙处理器（基于Zen 4架构）。
- 8 (第二位数字)：性能级别，数字越大（6-9为发烧级，1-4为主流级），性能越强。
- 00 (后两位数字)：通常为00或50，用于区分型号。
- X3D：后缀，代表不同特性。

**性能级别**

- Ryzen 3：入门级
- Ryzen 5：中端级
- Ryzen 7：高端级
- Ryzen 9：旗舰级

**常见后缀**

|后缀 | 含义 |
|-----|------|
| X   | 高性能版本 |
| G   | 集成显卡版本 |
|X3D | 三维缓存版本 |

**Other 版本**

AMD 推出过翼龙、速龙、推土机等系列的 CPU，但这些 CPU 已经不再生产，建议不要进行挑选。

### Intel

**主要特点**

单核性能强劲、存量市场大，兼容性好。

近段时间来 CPU 问题较多，导致市场反响严重。

如未来升级，主板需要进行更换。

**旧版本命名**

Intel 的 CPU 命名和 AMD 类似，但有一些不同之处。以 Intel Core i7-12700K 为例：
- Core：品牌名称。
- i7：产品定位，数字越大性能越强（i3 → i5 → i7 → i9）。
- 12 (第一位数字)：代数，代表第12代酷睿处理器（基于Alder Lake架构）。
- 700 (第二位数字)：性能级别，数字越大（00-99为发烧级，01-49为主流级，50-99为入门级），性能越强。
- K：后缀，代表可超频版本。

**最新版本命名**

Intel 最新的 CPU 统一命名为酷睿 Ultra 系统，去除 i 系列的命名方式，以 Intel Core Ultra 9 265K 为例：

- Core Ultra：品牌名称。
- 9：产品定位，数字越大性能越强（Core Ultra 3 → 5 → 7 → 9）。
- 265 (第一位数字)：代数，代表第26代酷睿处理器（基于Meteor Lake架构）。
- K：后缀，代表可超频版本。

**后缀（不包含笔记本）**

|后缀 | 含义 |
|-----|------|
| K   | 可超频版本 |
| KF  | 可超频版本，无集成显卡 |
| F   | 无集成显卡版本 |
|KS| 性能增强版本 |

### 总结

CPU 的命名规则虽然复杂，但可以总解决第一位数字表示产品定位，第一位或第二位数字表示代数，后缀表示特性。其余数字表示性能级别。

## CPU 选购建议

### 避坑指南

1. 不要选择宣传的 `i7 级` CPU，一般是洋垃圾。

### 品牌选择

在选择 CPU 时，AMD 和 Intel 是目前市场上最主要的两大品牌。AMD 的 Ryzen 系列和 Intel 的 Core 系列都提供了高性能的处理器。选择哪个品牌主要取决于个人偏好和具体需求。

两家并无决定性的选择，AMD 在多核性能和性价比方面表现更佳，而 Intel 在单核性能和兼容性方面更强。

### 使用需求

**日常使用**

选择 R3、i3 或 R5、i5 系列的 CPU 即可满足日常办公、上网和娱乐需求。

尤其建议选择附带集成显卡的 CPU，这样可以省去独立显卡的成本。

**主流游戏**

选择 R5、i7 或 i5、i7 系列的 CPU，可以满足大部分主流游戏的需求。尤其是配合独立显卡，可以获得更好的游戏体验。

建议选择 AMD 的 X3D 系统的 CPU，这样可以获得更好的游戏性能。

**内容创作**

视频剪辑、3D 渲染等内容创作任务需要更高的 CPU 性能。建议选择 R7、i9 或更高端的 CPU，这些处理器通常具有更多的核心和更高的主频，能够更好地处理多线程任务。

### 其他参考

**散热器**

CPU 的散热器是确保 CPU 在高负载下正常运行的重要组件。选择合适的散热器可以提高 CPU 的稳定性和性能。

i5 和 R5 系统的 CPU 推荐使用风冷散热器，i7 和 R7 系统的 CPU 推荐使用水冷散热器。

其中 AMD 处理器附带的散热器通常性能较好，用户可以根据实际需求选择是否更换。

**主板兼容性**

在选择 CPU 时，需要确保主板与 CPU 的兼容性。不同品牌和型号的 CPU 可能需要不同的主板芯片组。

**电源要求**

CPU 的功耗会影响电源的选择。确保电源能够提供足够的功率，以满足 CPU 和其他组件的需求。但一般 CPU 功耗较低，需要结合显卡的功耗来选择电源。

**内存兼容性**

主要的内存类型是 DDR4 和 DDR5。选择 CPU 时，需要确保主板支持所选内存类型。
