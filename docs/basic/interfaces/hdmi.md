# HDMI

## 简介

HDMI（High-Definition Multimedia Interface）是一种用于传输音频和视频信号的数字接口，广泛应用于电视、显示器、投影仪等设备。

## 主要版本

- HDMI 1.4：支持最高 4K@30Hz
- HDMI 2.0：支持最高 4K@60Hz
- HDMI 2.1：支持最高 10K@120Hz，刷新率和带宽大幅提升

## 型号与接口

- 标准 HDMI（Type A）
- Mini HDMI（Type C）
- Micro HDMI（Type D）

## 特性

- 支持音频回传通道（ARC/eARC）
- 支持 HDR、Dolby Vision、3D 等高级特性
